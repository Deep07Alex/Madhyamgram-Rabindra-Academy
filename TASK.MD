1. To change Colors, Spacing, and Global Styles

File to edit: 

client/src/index.css
What to do: Open this file to change the main colors (like the blue primary color), background colors, text sizes, and the overall layout design of the sidebars, dashboards, and login page.

2. To change Structure, HTML tags, and where things are placed

Folder to edit: client/src/pages/
Files inside: AdminDashboard.tsx, TeacherDashboard.tsx, StudentDashboard.tsx, Login.tsx
What to do: Open these files if you want to move a button to a new position, add new text, change the icons, or completely rearrange the layout (like moving the header to the bottom). These files contain the actual HTML-like code (called JSX).
